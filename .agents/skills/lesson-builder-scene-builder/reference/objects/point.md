# point

Static labeled point in 3D space.

```json
{"type":"point","position":[2,1,0],"color":"#ffffff","size":8,"label":"$(2,1)$"}
```

| Field | Default | Description |
|-------|---------|-------------|
| `position` | required | `[x,y,z]` in data space |
| `color` | `"#ffffff"` | Hex or `[r,g,b]` |
| `size` | `6` | Point size (pixels) |
| `label` | none | Supports LaTeX |
