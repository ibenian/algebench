Vision

Mathematics is a deeply visual discipline, yet most tools treat it as static text on a page. AlgeBench bridges the gap between symbolic reasoning and spatial intuition — turning abstract equations into explorable, interactive scenes where students can see a proof unfold, manipulate a function in real time, and discover connections across mathematical domains. Our goal is to make mathematical understanding feel as natural as exploring a map.

Frequently Asked Questions

  • Can I use AlgeBench online right now?
    Not yet — there's no hosted version at the moment. To try it, clone the repository and run it locally. It takes about two minutes to set up.
  • What do I need to run it locally?
    Just a GEMINI_API_KEY environment variable. You can create one for free at Google AI Studio. We recommend making a dedicated key for AlgeBench so you can track API usage separately.
  • What does AlgeBench use the Gemini API for?
    It powers two core features: an AI tutor you can ask questions to that understands the full lesson context — giving you intelligent, relevant answers as you learn — and real-time text-to-speech so the tutor can speak to you naturally as you explore.
  • Does AlgeBench support other AI models?
    Not at the moment. We're prioritizing improving the learning experience before expanding model support. Multi-model compatibility is something we'll consider in the future.
